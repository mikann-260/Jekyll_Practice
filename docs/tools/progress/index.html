経過率計算

入力された開始時間と終了時間から、現在どれだけ経過したかを計算します。

  • 1年は365日、1カ月は31日として計算されます。
  • 開始日時よりも終了日時の方が後である必要があります。
  • 日付の入力は必須です。
  • 時間は未入力の場合、午前0時として扱われます。
